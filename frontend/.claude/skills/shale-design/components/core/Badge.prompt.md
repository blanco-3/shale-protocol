Compact status or label pill; use for tier tags, tx states (EXECUTED / PENDING), buffer health, or mono codes.

```jsx
<Badge tone="apex" dot>First-loss</Badge>
<Badge tone="positive">Healthy</Badge>
<Badge tone="neutral" mono>0x3989…22DC</Badge>
```

Tones: `neutral`, `ink`, `core`/`seam`/`apex`, `positive`/`warning`/`danger`. `dot` adds a status dot, `outline` for hollow, `mono` for codes.
