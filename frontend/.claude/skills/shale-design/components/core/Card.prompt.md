Surface container for any grouped content; optional tier-keyed top edge makes it read as a labelled rock stratum.

```jsx
<Card accent="apex" strataEdge interactive>
  <h3>APEX</h3>
  <p>Leveraged residual yield.</p>
</Card>
<Card surface="ink">Dark feature panel</Card>
```

Surfaces: `paper` (default), `raised`, `sunken`, `ink`. `accent` adds a tier-colored top edge; `strataEdge` makes it a banded strata stripe; `interactive` adds hover lift.
