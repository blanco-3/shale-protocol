Primary action button — weighted, square-ish, geological-calm (no bounce); use for any deposit / connect / nav CTA.

```jsx
<Button tone="apex" iconRight="→">Deposit to APEX</Button>
<Button variant="outline">View Analytics</Button>
<Button variant="ghost" size="sm">Cancel</Button>
```

Variants: `solid` (ink fill, default), `outline` (hairline), `ghost` (text-only).
Tones: `default` (bedrock), `core` / `seam` / `apex` (tier accents), `accent` (copper-teal).
Sizes: `sm` / `md` / `lg`. Pass `fullWidth` for form CTAs.
