Metric tile for dashboards — eyebrow label, large mono value, optional caption and delta chip. Pair inside a `Card` or a stats grid.

```jsx
<StatTile label="Total TVL" value="$2,481,094" delta="+7.34%" sub="principal + yield" />
<StatTile label="APEX Buffer" value="18.4%" deltaTone="warning" sub="of total principal" />
```

Numbers always render in mono with tabular figures.
